## 2.8 Keyboard shortcuts

    Tab & 1 2 3     toggle edit mode and select vertex, edge or face mode
    ctrl + Tab      pie menu
    shift + space   play (depends)
    space           search (depends)
    ctrl + space    fullscreen window
    z               solid, wireframe ...
    q               quick favorites

    ctrl + a (PREFIX)
    -----------------
    panes
    -----
    o  swap
    %  vertical split
    "  horizontal split
    q  show pane number
    x  kill pane
    [space] toggle layouts
    { Move the current pane left
    } Move the current pane right
    z toggle pane zoom
    
    windows
    -------
    c  create window
    w  list windows
    n  next window
    p  previous window
    f  find window
    ,  name window
    &  kill window
    
    misc
    ----
    d  detach
    t  big clock
    ?  list shortcuts
    :  prompt
    
    outside commands
    ----------------
    tmux kill-session -t myname
